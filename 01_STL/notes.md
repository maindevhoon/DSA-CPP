# C++ STL Overview

The Standard Template Library (STL) is divided into four main parts:

1. **Algorithms**  
    Functions for processing data (e.g., sort, search, transform).

2. **Containers**  
    Data structures to store collections (e.g., vector, list, map).

3. **Functions**  
    Function objects (functors) and utilities for flexible operations.

4. **Iterators**  
    Objects to traverse containers and access elements.
