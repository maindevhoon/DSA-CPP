## Structure
Each chapter contains:
- `notes.md` — concepts, patterns, complexity cheatsheet
- `theory/` — hand-written code while learning
- `problems/` — LeetCode/GFG solutions named as `lc_<number>_<problem-name>.cpp`

## File Naming
`lc_1_two_sum.cpp` → LeetCode  
`gfg_kadane.cpp` → GeeksForGeeks

## Compile & Run
```bash
g++ -o out filename.cpp && ./out
```
